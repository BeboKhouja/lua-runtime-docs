# CreateNewGUI

**Type**: `function`

Creates a new GUI.

## Parameters

|Parameter      |Type       |Description                                           |
|---------------|-----------|------------------------------------------------------|
|narratedText   |string     |The string to be narrated when the GUI is displayed.|

## Returns
A table of this GUI.

## Example

```lua
Minecraft.CreateNewGUI("Lua Screen")
```
