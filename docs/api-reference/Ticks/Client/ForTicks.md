# ForTicks

**Type**: `function`

Run the specified function every tick.

## Parameters

|Parameter      |Type       |Description                    |
|---------------|-----------|-------------------------------|
|function       |function   |The function to run every tick.|

## Returns

Nothing.

## Example

```lua
Minecraft.Ticks.Client.ForTicks(function() 
    
end)
```
